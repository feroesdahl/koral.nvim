# koral.nvim

Neovim theme using colors inspired from coral reefs.

![screenshot of neovim koral theme](/images/screenshot.png)

## Installation

Install with your favorite package manager such as [folke/lazy.nvim](https://github.com/folke/lazy.nvim).

```lua
{
  "feroesdahl/koral.nvim",
  lazy = false,
  priority = 1000,
  opts = {},
}
```
